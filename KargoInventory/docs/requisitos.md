# Requisitos do Sistema - Kargo

## 1. Requisitos Funcionais

RF01 - Cadastrar produtos  
RF02 - Editar produtos  
RF03 - Registrar entrada de estoque  
RF04 - Registrar saída de estoque  
RF05 - Visualizar estoque atual  
RF06 - Listar histórico de movimentações  

## 2. Regras de Negócio

RN01 - O estoque não pode ser negativo  
RN02 - Toda movimentação deve ser registrada  
RN03 - Produto deve possuir categoria  

## 3. Requisitos Não Funcionais

RNF01 - Interface web responsiva  
RNF02 - Persistência em banco SQLite  
RNF03 - Tempo de resposta adequado  
RNF04 - Código modular e organizado  

## 4. Usuários do Sistema

- Administrador
- Operador de estoque
